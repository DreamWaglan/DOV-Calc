## 推荐文章类型

- 公式说明：解释计算过程、取整规则和适用边界。
- 数据记录：记录版本更新后的数值修正。
- 使用指南：说明某个计算器适合解决什么问题。
- 结论整理：把常见养成、资源或配队结论沉淀成可检索文章。
